# meterkast-dns

A local, external, Use-Pull resolver for smart-home device identity --
see README.md for the full design.

- [Resolved Names](/screens/resolved) -- live `dns`/`mdns` resolution results
- [All Devices](/screens/devices) -- every playlist entry, resolved or not
- [Log](/screens/logs) -- the backend's own recent activity, live

The classic single-table live view (no sidebar, no markdown pages) is
still available at [/table](/table). BLE/USB/HID scanning is at
[/web-scan](/web-scan).
